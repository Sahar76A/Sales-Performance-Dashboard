# Power Query Parameter Configuration

## Parameter Name
`pDataPath`

## Type
Text

## Purpose
Used to dynamically reference the local path of the Superstore CSV dataset.
This allows the solution to remain portable without hardcoding file paths.

## Example Value
C:\Users\<your-username>\Downloads\Sales-Performance-Dashboard\data\Sample - Superstore.csv

## Why This Matters
Using a parameter:
- Improves portability
- Avoids hard-coded paths
- Follows enterprise ETL best practices
